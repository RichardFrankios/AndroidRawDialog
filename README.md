# AndroidRawDialog
> 本项目主要介绍了Android原生的Dialog使用方式.主要包括以下几部分.
```
  1. 普通提示Dialog.
  2. 列表Dialog.
  3. 单选列表Dialog.
  4. 多选列表Dialog.
  5. 自定义ContentView Dialog.
  6. 自定义TitleView Dialog.
```
